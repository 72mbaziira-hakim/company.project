1. Web Design and Development
HTML/CSS/JavaScript: These are the building blocks of web development. HTML is used for structure, CSS for styling, and JavaScript for functionality.
React.js: A JavaScript library for building user interfaces, often used in single-page applications.
Next.js: A React framework that enables server-side rendering and generating static websites for React based web applications.
WordPress: A content management system (CMS) written in PHP. It’s often used for blog websites, but can be used for all kinds of websites.
2. Networking Services
Cisco technologies: Cisco provides a wide range of hardware and software solutions for networking.
Wireshark: A network protocol analyzer that lets you capture and interactively browse the traffic running on a computer network.
Nmap: A network scanner tool used for network discovery and security auditing.
3. Cybersecurity Services
Python: Often used in cybersecurity for writing scripts to automate tasks.
Kali Linux: A Linux distribution used for penetration testing and digital forensics.
Metasploit: A penetration testing framework that makes hacking simple. It’s used to test the defenses of your own network.
Wireshark: Also used in cybersecurity for network analysis and troubleshooting
